Music (Title)Bar

One of my all-time favorite WinAmp plugins has been Winamp title bar control. It creates a small set of controls in the corner of the currently active window and also displays the title & time.

MusicBar is a port of that plugin to TCMP and foobar2000. It's been tested under Windows XP. It may run run under Win9x but I cannot guess the results.

Screenshot:

The S button runs the play command Random in foobar2000 and in TCMP it switches the Shuffle On/Off.

Download:
To install the zipped plugin's, extract the zip to the plugin folder of the player and it should load the next time you start the player. To uninstall, simply delete the plugin.
